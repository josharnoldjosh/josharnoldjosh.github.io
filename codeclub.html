Scratch

Create a scene using Scratch. It must have:

- At least one animation

- At least one moving object

- A background

- A sound

- Text

- A character

An example could be an animation with different characters interacting with each other.

HTML

Create a story using HTML. It must have:

- Use of different colours

- Text

- A picture

- A p, h1, h2, and b tag

- Different coloured text

- Use of aligning text / divs

- CSS

An example could be a webpage with different paragraphs where you tell a story with images and different colours.

Python

Create a scene using python. It must have:

- Use the 'import turtle' to make moving turtles

- If statements

- Use the 'from random import randint' to generate random numbers

- A Loop

- an input to receive text

An example could be a chat bot that uses the 'text = input('replace me') and then 'if (text == 'yes'): else'. Remeber python requires text to be indented.
